# 카카오톡 플랫폼 설정 최종 확인 가이드

## 현재 상황
- ✅ 스크랩 결과: Vercel URL 정상
- ✅ 개발자 도구: Vercel URL 정상
- ✅ 코드: Vercel URL 사용
- ❌ 카카오톡 링크 클릭: GitHub로 이동

## 핵심 원인
카카오톡 개발자 콘솔의 **플랫폼 설정**에서 GitHub 도메인이 여전히 등록되어 있을 가능성이 높습니다.

## 즉시 확인할 항목

### 1. 카카오톡 개발자 콘솔 - 플랫폼 설정 (가장 중요)

1. **https://developers.kakao.com** 접속
2. 로그인 후 **내 애플리케이션** 선택
3. **앱 설정** → **플랫폼** 메뉴 클릭
4. **Web 플랫폼** 섹션 확인:
   - **사이트 도메인** 목록을 **자세히** 확인
   - `gusehd66.github.io`가 **어디에도 없어야 함**
   - `wedding-card-eight-kappa.vercel.app`가 **반드시 있어야 함**
5. 만약 GitHub 도메인이 있다면:
   - **완전히 삭제**
   - **저장** 버튼 클릭
6. 만약 Vercel 도메인이 없다면:
   - **추가**
   - **저장** 버튼 클릭

### 2. 카카오톡 개발자 콘솔 - 앱 도메인 설정

1. **앱 설정** → **일반** 메뉴
2. **앱 도메인** 확인
3. GitHub 도메인이 있다면 **삭제**
4. Vercel 도메인 추가 (필요시)

### 3. 카카오톡 개발자 콘솔 - JavaScript 키 확인

1. **앱 설정** → **앱 키** 메뉴
2. **JavaScript 키** 확인: `51ce90a5a85c4e81932e03038c1559be`
3. 코드의 JavaScript 키와 일치하는지 확인

### 4. 카카오톡 앱 완전 초기화

1. **카카오톡 앱 완전 종료**
   - 백그라운드에서도 제거
   - 또는 기기 재시작

2. **카카오톡 앱 재시작**

3. **새로운 링크로 공유**
   - Vercel 사이트에서 "카카오톡 공유하기" 클릭
   - **이전에 공유한 링크는 모두 삭제**

4. **새로 공유한 링크 클릭**
   - 어떤 URL로 이동하는지 확인

## 추가 확인사항

### 카카오톡이 링크를 크롤링할 때

카카오톡은 링크를 크롤링할 때:
1. 전달된 URL을 크롤링
2. Open Graph 메타 태그 확인
3. `og:url`이 있으면 그 URL로 리다이렉트

**하지만**, 카카오톡 개발자 콘솔의 플랫폼 설정에서 GitHub 도메인이 등록되어 있으면, 카카오톡이 내부적으로 다른 URL을 사용할 수 있습니다.

### Vercel 배포 확인

1. Vercel 대시보드에서 최신 배포 확인
2. 배포된 코드가 최신인지 확인
3. 빌드 로그에서 에러 확인

## 최종 해결 방법

만약 위의 모든 방법을 시도했는데도 문제가 지속된다면:

1. **카카오톡 개발자 콘솔에서 앱 삭제 후 재생성** (극단적 방법)
2. **새로운 JavaScript 키로 재설정**
3. **코드에서 JavaScript 키 업데이트**
