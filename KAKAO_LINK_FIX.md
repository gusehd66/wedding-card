# 카카오톡 링크 문제 해결 가이드

## 문제
카카오톡 공유 링크를 클릭했을 때 GitHub Pages로 리다이렉트되는 문제

## 해결 방법

### 1. 카카오톡 개발자 콘솔에서 링크 검증 및 캐시 클리어

1. **카카오톡 개발자 콘솔 접속**
   - https://developers.kakao.com 접속
   - 로그인 후 내 애플리케이션 선택

2. **링크 검증 도구 사용**
   - 좌측 메뉴에서 "도구" → "링크 검증" 클릭
   - URL 입력: `https://wedding-card-eight-kappa.vercel.app/`
   - "검증" 버튼 클릭
   - "캐시 삭제" 버튼 클릭 (있다면)

3. **앱 설정 확인**
   - "앱 설정" → "플랫폼" 메뉴
   - "Web 플랫폼" 등록 확인
   - 사이트 도메인: `wedding-card-eight-kappa.vercel.app` 확인

### 2. 카카오톡 앱에서 링크 다시 공유

1. Vercel 배포 사이트에서 "카카오톡 공유하기" 버튼 클릭
2. 새로운 링크로 공유 (이전에 공유한 링크는 삭제)
3. 새로 공유한 링크 클릭하여 확인

### 3. 카카오톡 앱 재시작

- 카카오톡 앱을 완전히 종료 후 다시 실행
- 또는 기기 재시작

### 4. 코드 확인

현재 코드는 이미 Vercel URL로 고정되어 있습니다:
- `src/utils/kakao.js`에서 `https://wedding-card-eight-kappa.vercel.app` 사용

### 5. 추가 확인사항

- 브라우저에서 직접 `https://wedding-card-eight-kappa.vercel.app/` 접속 확인
- Open Graph 메타 태그가 올바른지 확인 (개발자 도구 → Elements)
- 네트워크 탭에서 리다이렉트가 발생하는지 확인

## 참고

카카오톡은 링크 정보를 캐시하므로, 변경사항이 반영되기까지 시간이 걸릴 수 있습니다.
캐시 삭제 후 몇 시간에서 하루 정도 기다려야 할 수도 있습니다.
